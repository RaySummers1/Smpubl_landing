# Smpubl_landing
# 🎵 SMPUBL℠ Music Licensing Landing Page

This is the official landing page for SMPUBL℠ — a smart music publishing platform that empowers artists and rights holders with transparent licensing tools.

---

## 🚀 Project Overview

This landing page allows artists to:
- Learn about SMPUBL℠ publishing services
- Submit their information for licensing and royalty registration
- View genre-specific branding and UI

---

## 🧱 Folder Structure

| Folder        | Purpose |
|---------------|---------|
| `public/`     | Static HTML, CSS, and assets |
| `src/`        | Frontend JavaScript logic |
| `data/`       | Sample artist data |
| `backend/`    | Python backend for form handling |
| `.env`        | Environment variables |
| `package.json`| Frontend dependencies |

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript (Vanilla)
- Python (Flask or FastAPI)
- CSV for sample data
- Optional: Cloudinary or Firebase for asset storage
